Blisk a web browser to help building your frontend the right way

Blisk is the new kid on the block to help web developers getting things done right. Blisk is a web browser that has three frames on its interface.

  • The left frame display a range of Mobile device that you can choose.
  • The middle frame display how a given website will look like on the chosen device (the left frame)
  • the right frame displays the website as you could see it in your day-to-day browser like Firefox or Chrome.

For sure Firefox and Google Chrome offers many addons or extensions that you can toggle to view how the website will look like in an Iphone but you have to toggle or to enable those addons/extensions. By default Blisk come with similar extension already enable.

Blisk is based on Chromium the open-source webroswer which Google Chrome is based on. So that means that every extension that you like in Chrome you can install it on Blisk. Here is an overview of Blisk functionnalities.

Blisk is more than a fancy browser, as its core it should be the first choice for frontend developpers to test the layout of their web application.
